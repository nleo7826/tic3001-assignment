# Demo for Redis Cache

```
Start Redis server - redis-server
Start Redis client - redis-cli
Start Express server - nodemon server
```

```
Inside redis-cli:
    flushall - delete all the keys of existing db
    keys * - return all keys
    get <key> - show value of key
```

```
Inside postman:
    Retrive data endpoint - GET - localhost:8080/photos
```
